# Giphy

**Quickly insert GIFs and MEMEs into Roam without leaving the editor.**

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/RoamJS/giphy)

## Usage

Type `/gif` in any block and select `Giphy: Insert GIF` from the slash menu. This opens the GIF picker overlay.

In the picker:

- Type to search GIPHY results (or browse trending by default).
- Use arrow keys or hover to select a GIF.
- Press `Enter` (or click) to insert it at the cursor.
- Press `Esc` to close.

The extension inserts markdown image syntax directly into your block, for example:

```markdown
![Loop Jump GIF by sheepfilms](https://media2.giphy.com/media/gKfNj8cYeGN63bLRkF/giphy.gif?cid=d3ed119egqzsfvzi63mxlmvcs8qrd9cr5jdx40p39idxvazc&rid=giphy.gif)
```
